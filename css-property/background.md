background
다른 background 속성들을 하나의 선언으로 설정한다.

설정할 수 있는 속성들
background-color: 사용할 배경색 명시
background-image: 사용할 하나 혹은 여러 개의 배경 이미지 명시
background-position: 배경 이미지들의 position 명시
background-size: 배경 이미지들의 크기 명시
background-repeat: 어떻게 배경 이미지들을 반복할 지 명시
background-origin: 배경 이미지들의 위치 영역을 명시
background-clip: 배경 이미지를 칠할 영역을 명시
background-attachment: 배경이미지가 고정되있는지 아니면 다른 페이지와 스크롤 될 지 결정

속성들 중 몇개가 생략되어도 상관없음.

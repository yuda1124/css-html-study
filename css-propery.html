display

대상 엘리먼트에 사용되는 렌더링 박스의 유형을 결정해주는 css 속성.
default 값은 HTML 사양 또는 브라우저 / 사용자 기본 스타일 시트에서 가져온다.
display property는 키워드 값을 사용하여 지정되며 키워드 값은 6 가지 카테고리로 분류된다.

  • 1. display-outside

    엘리먼트에 의해 생성 된 박스의 외부 디스플레이 타입을 지정하여 엘리먼트가 어떻게 부모 formatting context에 참여 하는지를 지시한다.

    Property Values

    • block: 이 Element는 block element box를 만든다.

    • inline: 이 Element는 하나 혹은 이상의 inline element box를 만든다.

  • 2. display-inside

    엘리먼트에 의해 생성 된 박스의 내부 디스플레이 타입을 지정하며, 그 내용이 박스 안에 어떻게 배치되는지 지시한다.

    Property Values

    • table: 이 Element는 <table> HTML Element처럼 동작합니다. block-level box를 정의합니다.

    • flex: 이 Element는 block element 처럼 동작하고 flexbox 모델에 따라 내용을 배치합니다.

    • grid: 이 Element는 block element 처럼 동작하고 grid 모델에 따라 내용을 배치합니다.

  • 3. display-listitem

    이 Element는 내용을 위한 block box와 개별 list-item의 inline box를 만듭니다.
    명시된 display inside 값이 없으면 box의 inner display-type은 flow가 됩니다.
    명시된 display outside 값이 없으면 box의 outer display-type은 block이 됩니다.

  • 4. display-internal

    이 섹션에서는 특정 레이아웃 모드 내에서만 의미있는 "내부" display 값을 정의합니다.

    Property Values

    • table-row-group: <tbody> HTML Element 처럼 동작한다.

    • table-header-group: <thead> HTML Element 처럼 동작한다.

    • table-footer-group: <tfoot> HTML Element 처럼 동작한다.

    • table-row: <tr> HTML Element 처럼 동작한다.

    • table-cell: <td> HTML Element 처럼 동작한다.

    • table-column-group: <colgroup> HTML Element처럼 동작한다.

    • table-column: <col> HTML Element처럼 동작한다.

    • table-caption: <caption> HTML Element처럼 동작한다.

  • 5. display-box

    이 값은 요소가 display box를 생성하는지 여부를 정의합니다.

    • none: 레이아웃에 영향을 미치지 않도록 요소의 표시를 끈다.

  • 6. display-lagacy

    CSS2에서는 display property가 하나의 keyword만 사용되었기 때문에 사용했던 value들

overflow

해당 엘리먼트의 블록 레벨 컨테이너(block-level container)보다 내용(content)이 너무 클 때 내용을 잘라낼 지, 스크롤 바를 보여줄 지, 혹은 범위를 넘어가게 보여줄 지를 기술한다.

Property Values

  • visible(default): 내용이 잘리지 않음. 엘리먼트 박스의 범위를 넘어서 보여준다.

    여우가 나타난 것은 바로 그때였다. "안녕." 여우가 말했다. "안녕." 어린 왕자가 얌전히 대답하고 몸을 돌렸으나 아무것도 보이지 않았다. "난 여기 사과나무 밑에 있어." 좀 전의 그 목소리가 말했다. "넌 누구지? 넌 참 예쁘구나." 어린 왕자가 말했다. "난 여우야." 여우는 말했다. "이라 와서 나하고 놀자. 난 아주 슬프단다." 어린 왕자가 제의했다. "난 너하고 놀 수 없어. 나는 길들어져 있지 않거든." 여우가 말했다.

  • hidden: 내용을 자르고, 내용의 나머지는 보여주지 않는다.

    여우가 나타난 것은 바로 그때였다. "안녕." 여우가 말했다. "안녕." 어린 왕자가 얌전히 대답하고 몸을 돌렸으나 아무것도 보이지 않았다. "난 여기 사과나무 밑에 있어." 좀 전의 그 목소리가 말했다. "넌 누구지? 넌 참 예쁘구나." 어린 왕자가 말했다. "난 여우야." 여우는 말했다. "이라 와서 나하고 놀자. 난 아주 슬프단다." 어린 왕자가 제의했다. "난 너하고 놀 수 없어. 나는 길들어져 있지 않거든." 여우가 말했다.

  • scroll: 내용이 잘리지만 남은 내용을 볼 수 있게 하는 스크롤바가 생성된다. (항상 있는)

    여우가 나타난 것은 바로 그때였다. "안녕." 여우가 말했다. "안녕." 어린 왕자가 얌전히 대답하고 몸을 돌렸으나 아무것도 보이지 않았다. "난 여기 사과나무 밑에 있어." 좀 전의 그 목소리가 말했다. "넌 누구지? 넌 참 예쁘구나." 어린 왕자가 말했다. "난 여우야." 여우는 말했다. "이라 와서 나하고 놀자. 난 아주 슬프단다." 어린 왕자가 제의했다. "난 너하고 놀 수 없어. 나는 길들어져 있지 않거든." 여우가 말했다.

  • auto: 내용이 잘리면 남은 내용을 볼 수 있게 하는 스크롤바가 생성된다. (필요시 생성)

    여우가 나타난 것은 바로 그때였다. "안녕." 여우가 말했다. "안녕." 어린 왕자가 얌전히 대답하고 몸을 돌렸으나 아무것도 보이지 않았다. "난 여기 사과나무 밑에 있어." 좀 전의 그 목소리가 말했다. "넌 누구지? 넌 참 예쁘구나." 어린 왕자가 말했다. "난 여우야." 여우는 말했다. "이라 와서 나하고 놀자. 난 아주 슬프단다." 어린 왕자가 제의했다. "난 너하고 놀 수 없어. 나는 길들어져 있지 않거든." 여우가 말했다.

  • inherit: 부모 엘리먼트와 같은 값을 사용한다.
  • initial: default value를 사용한다.

visible (default)가 아닌 다른 값으로 overflow 속성 사용 시 새로운 블록 서식 문맥(block formatting context) 를 생성하게 된다.
overflow 속성이 효력을 갖기 위해서는 반드시 블록 레벨 컨테이너의 높이(height 또는 max height)가 설정되어 있거나, white-space가 nowrap으로 설정되어야 한다.

background

다른 background 속성들을 하나의 선언으로 설정한다.

  • 1. background-color

    사용할 배경색을 명시한다.

    Property Values
    • 유효한 색깔명 ex) background-color: red;
    • HEX 값 ex) background-color: #ff0000;
    • RGB 값 ex) background-color: rgb(255,0,0);

  • 2. background-image

    사용할 하나 혹은 여러 개의 배경 이미지를 명시한다. default로 image는 전체 공간에 반복된다.

    Property Values
    • url ex) background-image: url("star.png");

  • 3. background-position

    배경 이미지들의 position을 명시한다.

    example
    • background-position: center top;
    • background-position: 50% 50%;
    • background-position: 50px 100px;

  • 4. background-size

    배경 이미지들의 크기를 명시한다.

    Property Values
    • auto(default) 원본 사이즈 크기에 맞게
    • length width, height 순으로 정의. 하나의 값만 주어진다면 height를 auto로 적용

      background-size: 50px 50px;

    • percentage 부모 element의 percent로 정의. 하나의 값만 주어진다면 height를 auto로 적용

      background-size: 200% 200%;

    • cover image를 전체 컨테이너를 덮도록 resize한다.

      background-size: cover;

    • contain 배경 이미지의 크기를 조정하여 이미지가 완전히 보이도록 한다.

      background-size: contain;

  • 5. background-repeat

    어떻게 배경 이미지들을 반복할 지 명시한다.

    Property Values
    • repeat(default) 수직으로 수평으로 모두 반복. 잘 맞지 않는다면 마지막 이미지는 잘릴 수 있음

      background-repeat: repeat;

    • repeat-x 배경 이미지가 수평으로만 반복된다.

      background-repeat: repeat-x

    • repeat-y 배경 이미지가 수직으로만 반복된다.

      background-repeat: repeat-y;

    • no-repeat 배경이미지가 반복되지 않는다.

      background-repeat: no-repeat;

    • space 배경이미지가 잘리지 않을 최대한으로 반복된다. 처음과 마지막 이미지는 element 양쪽에 고정되며 공백은 이미지간에 분산된다.

      background-repeat: space;

    • round 배경이미지가 공간을 채우기 위해 반복되고 찌그러지거나 늘어난다.

      background-repeat: round;

  • 6. background-origin

    배경 이미지들의 시작 위치를 명시한다.
    background-attachment가 fixed일 때는 아무 효과가 없음.

    Property Values
    • padding-box (default) padding edge의 왼쪽 위 코너부터 배경이미지가 시작된다.

      background-origin: padding-box;

    • border-box border의 왼쪽 위 코너부터 배경이미지가 시작된다.

      background-origin: border-box

    • content-box content의 왼쪽 위 코너부터 배경이미지가 시작된다.

      background-repeat: content-box;

  • 7. background-clip

    배경이 요소내에서 어느 정도 확장되어야 하는지 명시한다.

    Property Values
    • border-box (default) border 뒤쪽으로 배경이 확장된다.

      background-clip: border-box;

    • padding-box border edge의 안쪽으로 배경이 확장된다.

      background-clip: padding-box

    • content-box context box의 edge 쪽으로 배경이 확장된다.

      background-clip: content-box;

  • 8. background-attachment

    배경이미지가 고정되있는지 아니면 다른 페이지와 스크롤 될 지 결정한다.

    Property Values
    • scroll (default) 배경 이미지는 page와 함께 scroll 된다.

      background-attachment: scroll;

    • padding-box 배경이미지는 page와 scoll되지 않는다.

      background-attachment: fixed;

    • local 배경이미지는 element의 contents와 함께 scroll 된다.

      background-attachment: local;

word-break

텍스트가 content box에서 overflow 될 때 브라우저가 줄바꿈을 삽입하여야 하는지를 명시한다.

Property Values

  • normal(initial): default 줄바꿈 규칙을 적용

    "Goodmorning,"saidthe fox. "Good morning,"thelittleprincerespondedpolitely,althoughwhenheturnedaroundhesawnothing. "Iamrighthere,"the voice said,"undertheappletree."

  • break-all: overflow를 방지하기 위해서 어떤 두 개의 문자 사이에도 줄바꿈을 적용한다. (한국어, 일본어, 중국어 text 제외)

    "Goodmorning,"saidthe fox. "Good morning,"thelittleprincerespondedpolitely,althoughwhenheturnedaroundhesawnothing. "Iamrighthere,"the voice said,"undertheappletree."

  • keep-all: 스페이스나 탭 또는 엔터로 띄운 단어를 기준으로 줄바꿈을 적용. 띄어쓰기가 없다면 줄 바꿈이 되지 않는다.

    "Goodmorning,"saidthe fox. "Good morning,"thelittleprincerespondedpolitely,althoughwhenheturnedaroundhesawnothing. "Iamrighthere,"the voice said,"undertheappletree."

word-wrap

텍스트 줄 바꿈 모드를 설정한다.

Property Values

  • normal: 기본값으로 내용(contents)가 컨테이너 경계를 초과한다.

    "Goodmorning,"saidthe fox. "Good morning,"thelittleprincerespondedpolitely,althoughwhenheturnedaroundhesawnothing."Iamrighthere,"thevoicesaid,"undertheappletree."

  • break-word: content가 다음 줄로 넘어가고 필요한 경우 단어 줄 바꿈이 발생.

    "Goodmorning,"saidthe fox. "Good morning,"thelittleprincerespondedpolitely,althoughwhenheturnedaroundhesawnothing."Iamrighthere,"the voice said,"undertheappletree."

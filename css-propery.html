display

overflow

해당 엘리먼트의 블록 레벨 컨테이너(block-level container)보다 내용(content)이 너무 클 때 내용을 잘라낼 지, 스크롤 바를 보여줄 지, 혹은 범위를 넘어가게 보여줄 지를 기술한다.

  • visible(default): 내용이 잘리지 않음. 엘리먼트 박스의 범위를 넘어서 보여준다.

    서해 상에서 해기차(바다와 상층 공기의 온도 차이)에 의해 생성된 구름대의 영향으로 전라도와 제주도, 충남 서해안에는 계속해서 '눈 폭탄'이 떨어지고 있다. 오후 4시 현재 적설량은 목포 16.9㎝, 광주 13.6㎝, 홍성 10.5㎝, 영광 22.0㎝, 부안 19.0㎝, 고창 18.0㎝, 어리목(제주 산지) 40.2㎝, 새만금 27.7㎝, 함평 20.2㎝, 무안 17.8㎝, 김제 14.8㎝ 등이다.

  • hidden: 내용을 자르고, 내용의 나머지는 보여주지 않는다.

    서해 상에서 해기차(바다와 상층 공기의 온도 차이)에 의해 생성된 구름대의 영향으로 전라도와 제주도, 충남 서해안에는 계속해서 '눈 폭탄'이 떨어지고 있다. 오후 4시 현재 적설량은 목포 16.9㎝, 광주 13.6㎝, 홍성 10.5㎝, 영광 22.0㎝, 부안 19.0㎝, 고창 18.0㎝, 어리목(제주 산지) 40.2㎝, 새만금 27.7㎝, 함평 20.2㎝, 무안 17.8㎝, 김제 14.8㎝ 등이다.

  • scroll: 내용이 잘리지만 남은 내용을 볼 수 있게 하는 스크롤바가 생성된다. (항상 있는)

    서해 상에서 해기차(바다와 상층 공기의 온도 차이)에 의해 생성된 구름대의 영향으로 전라도와 제주도, 충남 서해안에는 계속해서 '눈 폭탄'이 떨어지고 있다. 오후 4시 현재 적설량은 목포 16.9㎝, 광주 13.6㎝, 홍성 10.5㎝, 영광 22.0㎝, 부안 19.0㎝, 고창 18.0㎝, 어리목(제주 산지) 40.2㎝, 새만금 27.7㎝, 함평 20.2㎝, 무안 17.8㎝, 김제 14.8㎝ 등이다.

  • auto: 내용이 잘리면 남은 내용을 볼 수 있게 하는 스크롤바가 생성된다. (필요시 생성)

    서해 상에서 해기차(바다와 상층 공기의 온도 차이)에 의해 생성된 구름대의 영향으로 전라도와 제주도, 충남 서해안에는 계속해서 '눈 폭탄'이 떨어지고 있다. 오후 4시 현재 적설량은 목포 16.9㎝, 광주 13.6㎝, 홍성 10.5㎝, 영광 22.0㎝, 부안 19.0㎝, 고창 18.0㎝, 어리목(제주 산지) 40.2㎝, 새만금 27.7㎝, 함평 20.2㎝, 무안 17.8㎝, 김제 14.8㎝ 등이다.

  • inherit: 부모 엘리먼트와 같은 값을 사용한다.
  • initial: default value를 사용한다.

visible (default)가 아닌 다른 값으로 overflow 속성 사용 시 새로운 블록 서식 문맥(block formatting context) 를 생성하게 된다.
overflow 속성이 효력을 갖기 위해서는 반드시 블록 레벨 컨테이너의 높이(height 또는 max height)가 설정되어 있거나, white-space가 nowrap으로 설정되어야 한다.

background

다른 background 속성들을 하나의 선언으로 설정한다.

  • background-color: 사용할 배경색을 명시한다.
  • background-image: 사용할 하나 혹은 여러 개의 배경 이미지를 명시한다.
  • background-position: 배경 이미지들의 position을 명시한다.
  • background-size: 배경 이미지들의 크기를 명시한다.
  • background-repeat: 어떻게 배경 이미지들을 반복할 지 명시한다.
  • background-origin: 배경 이미지들의 위치 영역을 명시한다.
  • background-clip: 배경 이미지를 칠할 영역을 명시한다.
  • background-attachment: 배경이미지가 고정되있는지 아니면 다른 페이지와 스크롤 될 지 결정한다.

white-space

해당 엘리먼트 안의 공백이 어떻게 처리될지를 나타내는데 사용된다.

  • normal: 연속된 공백이 하나로 병합. 소스 안의 줄바꿈 문자는 다른 공백 문자와 같이 취급된다. 필요에 따라 줄 바꿈함.

  • nowrap: 공백문자를 병합하지만, 텍스트 줄바꿈을 하지 않는다.

  • pre: 연속된 공백이 보존된다. 줄바꿈은 줄바꿈 문자나 < br > 요소에 의해서만 이뤄진다.

  • pre-wrap: 연속된 공백이 보존된다. 줄바꿈은 줄바꿈 문자, < br >, 줄 박스를 채우기 위해 필요에 따라 이뤄진다.

  • pre-line: 연속된 공백이 병합된다. 줄바꿈은 줄바꿈 문자, < br >, 줄 박스를 채우기 위해 필요에 따라 이뤄진다.

  • initial: default value를 사용한다.

word-break

텍스트가 content box에서 overflow 될 때 브라우저가 줄바꿈을 삽입하여야 하는지를 명시한다.

  • normal(initial): default 줄바꿈 규칙을 적용
  • break-all: overflow를 방지하기 위해서 어떤 두 개의 문자 사이에도 줄바꿈을 적용한다. (한국어, 일본어, 중국어 text 제외)
  • keep-all: 스페이스나 탭 또는 엔터로 띄운 단어를 기준으로 줄바꿈을 적용. 띄어쓰기가 없다면 줄 바꿈이 되지 않는다.

word-wrap

텍스트 줄 바꿈 모드를 설정한다.

  • normal: 기본값으로 내용(contents)가 컨테이너 경계를 초과한다.
  • break-word: content가 다음 줄로 넘어가고 필요한 경우 단어 줄 바꿈이 발생.
